# GCD 복잡도
# 1. 이 프로그램은 두 정수를 입력받아 최대공약수(GCD)를 계산하는 프로그램이다. 

# GCD 알고리즘 코드

int gcd(int a, int b)
{
	if (a < 0)
	{
		a = -a;
	}
	if (b < 0) {
		b = -b;
	}

	if (a == 0 && b == 0)
	{
		return 0;
	}

	int i;
	int result = 1;

	for (i = 1; i <= a && i <= b; i++)
	{
		if (a % i == 0 && b % i == 0)
		{
			result = i;
		}
	}
	
	return result;
}

# 2. 시간복잡도
for (i = 1; i <= a && i <= b; i++) 이 반복문은 i = 1부터 a 와 b 더 작은 값까지 반복된다. 따라서 박복 횟수는 최대 min(a,b)이다.
ex) gcd(10,20) -> 최대 10번 실행 ,  gcd(100,120) -> 최대 100번 실행
결론: 두 수 중 작은 값이 커지면 반복 횟수가 비례해서 증가한다

# 3.Big-O 계산
작은 값에 비례해서 반복 횟수가 증가하므로 이 알고리즘의 Big-O는 n = min(a,b) 라고 하면 O(n) 이라고 할 수 있다.

# 결론
이 알고리즘은 입력값이 커질수록 연산 횟수도 선형적으로 증가하는 알고리즘이다.
